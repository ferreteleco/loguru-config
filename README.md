# README file for Loguru config parser utility

Initial version of a simple yet flexible config parser for Loguru intialization via .ini file.

Currently, it supports:

- Colorization
- Formating
- Sink selection
- Filtering*
- Queuing
- Serializing

*Wihtout support for lambda expressions
